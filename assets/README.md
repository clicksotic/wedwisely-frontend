# Assets Directory

This directory contains the app icons and splash screen images for the WedWisely app.

Required images:
- `icon.png` - App icon (1024x1024 recommended)
- `splash.png` - Splash screen (1242x2436 recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 recommended)
- `favicon.png` - Web favicon (32x32 recommended)

For now, placeholder images have been created. Replace these with your actual app branding images.
